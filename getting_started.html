Getting Started

Overview

Since Arkito is an open source game engine, the setup is a relatively simple process of downloading the repository and working directly in the project. When creating your game, do not rename or move the /arkiGame/ package. Only packages with the prefix “sample” can be moved, deleted, and modified safely. The /arkiGame/scenes/ package cannot be renamed or removed, however, its contents can be modified.

Steps Summary

  • Download the repository
  • Setup the app configuration
  • Create your default scene

Downloading the repository

Refer to a tutorial on the basics of GitHub if this step is difficult.

  1. Go to the Arkito GitHub Repository by L0raxeo
  2. Download the repository either directly or from a fork
  3. Open the project in IntelliJ

Setting up the App Configuration

the Arkito game engine will automatically scan the main /arkiGame/ package and create a list of all the classes. Then, it will find the class with the @AppConfig annotation in the list.

  1. Create a class to store the App Configuration
    1. The class name does not matter
    2. The class must be in the /arkiGame/ package
  2. Add the App Configuration annotation
    1. Specify windowTitle - title of your application window
    2. Specify windowWidth - width of your application window
    3. Specify windowHeight - height of your application window
    4. Specify resizeable - determines whether your application window can be resized
