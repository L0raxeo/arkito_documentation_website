Overview

Game Objects are containers for a collection of components and for positional attributes. Update, render, start, and die methods are all handled by the scene along with all of its inherited components.

UID System

Each game object is identified with a unique identifier associated with the number of game objects created at the time of its instantiation. It contains a string name, however, it should be considered to be an unsafe identifier since multiple game objects can have the same name.
