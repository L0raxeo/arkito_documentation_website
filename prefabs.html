Prefabs

Overview

Prefabricated game objects allow you to create a blueprint for instantiating your game objects without actually instantiating it. Thus, you can create game objects with one simple line of code. The static prefabs class has two static methods for generating game objects. Their parameters pass in attributes for the game object including its name, position, size, rotation, and components.
