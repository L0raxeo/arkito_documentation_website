Mouse Motion

Overview

Arkito’s Mouse Manager class handles mouse motion and mouse position for you by encapsulating the AWT library’s Mouse Motion Listener interface. If you are unfamiliar with the system, it is recommended that you take a look at some basic documentation. The user interacts with the getters of the Mouse Manager class. See the Screen vs. Graph vs. World Coordinates documentation page for information on coordinate systems.
