Overview

Adding collisions to your game object is as simple as adding the box-bounds component.

Simulating Collisions

The box-bounds component will look at the rigidbodyâ€™s planned path and determine whether it will collide with any objects. If it does, it will send back any collisions it detects to the rigidbody to be managed.

On Collision

The game object will invoke the onCollision method within all of its components if a box-bounds component is attached to it with a rigidbody. It passes through the parameters a collision object that encapsulates holds the game object being collided with, the game object colliding, and the collision type.
