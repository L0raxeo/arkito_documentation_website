Physics

Overview

The physics component requires the rigidbody component for it to function. It adds gravitational acceleration to the rigidbody.

Requirements

Must have a rigidbody component already attached to the game object. Notice line 29 and 30 of the example below. The rigidbody component is added prior to the physics component. When the physics component’s start method is called, it will automatically search for its parent game object’s rigidbody. If it is not found, it may cause runtime errors.
